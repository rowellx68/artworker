# artworker

> Artwork downloader.


## Dev

```
npm install && bower install
```

### Run

```
npm start
```

### Build

```
// for all platforms
npm run build

// for osx
npm run build-osx

// for windows
npm run build-win
```

Builds the app for OS X [electron-packager](https://github.com/maxogden/electron-packager).
